Using this dashboard

  1. Confirm the four summary tiles (applicants, open jobs, shift slots, pending applications).
  2. Use Department load and Availability by day to see where demand and student time cluster.
  3. Open Candidate matches by job, select a job in the list, and read the ranked Score (skills and schedule fit).
  4. When a student changes profile or schedule on the Applicant view, the next auto-refresh or Refresh all updates metrics.
  5. Use Reload matches after changing the job only, without waiting for a full round trip.
